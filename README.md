# Currency Converter

This is a simple currency converter app. It allows users to convert between different currencies based on current exchange rates.

## Installation

No installation is required. Simply open the `index.html` file in your web browser to use the currency converter.

## Usage

1. Select the currency you want to convert from and to from the dropdown menus.
2. Enter the amount you want to convert.
3. Click the "Convert" button to see the converted amount.
